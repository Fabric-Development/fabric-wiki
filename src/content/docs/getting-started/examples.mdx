## [Simple Bar](https://github.com/Fabric-Development/fabric/tree/main/examples/bar)
This configuration demonstrates the usage of Fabric in a status bar example.
This serves as your perfect starting point for building a status bar on Hyprland, and it can be easily modified to adapt other compositors, X11, too!
![Simple Bar](/assets/examples/bar.webp)

## [Notifications Daemon](https://github.com/Fabric-Development/fabric/tree/main/examples/notifications)
This configuration teaches you how to interact with the [`Notifications` Service](../api/services/notifications.mdx) and create a notifications widget using it, this is the best example for whom looking to make a notifications center/daemon.
![Notification Popups](/assets/examples/notifications.webp)

## [Application Launcher](https://github.com/Fabric-Development/fabric/tree/main/examples/app-launcher)
This configuration demonstrates the proper way of implementing a fast application launcher.
![Application Launcher](/assets/examples/app-launcher.webp)

## [Icon Browser](https://github.com/Fabric-Development/fabric/tree/main/examples/icon-browser)

## [Bluetooth Discovery](https://github.com/Fabric-Development/fabric/tree/main/examples/bluetooth-widget)
A tool that makes use of the [`Bluetooth` Service](../api/services/bluetooth.mdx) to provide the user with Bluetooth devices and connections manager that is fully made using Fabric!
![Bluetooth Widget](/assets/examples/bluetooth.webp)
